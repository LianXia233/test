# 校外培训机构综合管理平台

## 产品定位

本项目定位为：**面向培训机构的 SaaS 教学管理系统**。

平台核心目标是帮助培训机构实现从招生到教学、从教务到财务的一体化数字管理，提升运营效率并降低管理成本。

## 核心能力

平台将重点支持以下能力：

1. **多角色协同**
   - 超级管理员
   - 机构管理员
   - 教师
   - 学员/家长
   - 财务人员

2. **多机构管理**
   - 支持连锁品牌下多校区统一管理
   - 支持机构级数据隔离与权限控制

3. **选课管理**
   - 课程发布、班级管理、排课与调课
   - 学员选课、退课、转班流程管理

4. **作业管理**
   - 教师布置作业与批改
   - 学员在线提交与完成进度跟踪

5. **成绩管理**
   - 成绩录入、统计分析、学习报告
   - 阶段性测评与成长档案沉淀

6. **收费管理**
   - 订单、收费、退款与对账
   - 收入报表与财务数据可视化

## 目标价值

- 对机构管理者：提升运营透明度与决策效率
- 对教师团队：降低教务事务成本，聚焦教学质量
- 对学员与家长：提升学习过程可见性与服务体验

## 建议的 MVP 范围

首期建议围绕“**选课 + 作业 + 成绩 + 收费**”四大模块建设可用闭环，并优先完成：

- 基础账号与角色权限体系
- 机构与校区管理
- 课程/班级与选课流程
- 作业发布、提交与批改
- 成绩录入与报告输出
- 收费、退款与财务统计
